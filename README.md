# Java Sudoku Solver

## Description
This is a Java-based Sudoku Solver that solves a given 9x9 Sudoku puzzle using
a backtracking algorithm and displays the solved puzzle.

## Key Features
- Solves 9x9 Sudoku puzzles
- Uses backtracking algorithm
- Prints solved Sudoku grid
- Simple and clean logic
- Beginner-friendly implementation

## Tech Stack
- Java
- Recursion
- Backtracking algorithm

## How to Run
1. Clone or download the repository
2. Compile the program:
   javac EasySudoku.java
3. Run the program:
   java EasySudoku

## Purpose
- Develop algorithmic thinking and debugging skills
- Showcase a project that demonstrates intermediate Java programming skills
- This project is developed to improve logical thinking and problem-solving skills using Java.
